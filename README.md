## 3.强调
```
*强调*(斜体) 
_强调_(斜体) 
**加重强调**(粗体) 
__加重强调__(粗体) 
***特别强调***(粗斜体) 
___特别强调___(粗斜体) 
```
效果：<br>
*强调*(斜体)<br>
_强调_(斜体)<br>
**加重强调**(粗体)<br>
__加重强调__(粗体)<br>
***特别强调***(粗斜体)<br>
___特别强调___(粗斜体)<br>
